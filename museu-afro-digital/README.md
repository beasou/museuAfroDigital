# Museu Afro Digital - Projeto Autoral

Projeto autoral do Módulo 3: site sobre memória e cultura afro-brasileira.

## Páginas

- `index.html` — página inicial com apresentação e destaque das coleções.
- `sobre.html` — contextualização do projeto e referências teóricas.
- `colecoes.html` — exemplo de coleções digitais (documentos, fotografias, áudio & vídeo).
- `contato.html` — formulário de contato ilustrativo.

## Tecnologias

- HTML5
- Bootstrap 4 (via CDN)
- Imagens locais na pasta `img/`

## Autoria

- Aluna: **Beatriz Regina de Souza**
- Disciplina: Desenvolvimento para Internet 1
- Avaliação: Módulo 3 – Museu Digital e Cultura Afro-Brasileira
